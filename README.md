# Wallpaper Switcher

A simple **wallpaper switcher for Rofi** written in **sh (POSIX shell)**.

This script allows you to quickly change wallpapers through a **Rofi menu**.

## Requirements

Make sure you have the following installed:

- rofi
- swww

## Setup

1. Clone this repository:

```bash
git clone https://github.com/Rotmayer24/Wallpaper-Switcher.git
cd Wallpaper-Switcher
chmod +x WallpaperSwitcher.sh
./WallpaperSwitcher.sh
